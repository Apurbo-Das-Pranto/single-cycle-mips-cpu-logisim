# 4-Bit MIPS Processor (Logisim)

A custom 4-bit MIPS processor designed and simulated in Logisim Evolution for the CSE 210 Computer Architecture Sessional course at BUET.

## Project Information

* Course: CSE 210 – Computer Architecture Sessional
* Institution: Bangladesh University of Engineering and Technology (BUET)
* Group ID: 2
* Section: B2

## Features

* 4-bit ALU
* 8-bit Address Bus
* 4-bit Data Bus
* Custom MIPS Instruction Set
* Microprogrammed Control Unit
* Instruction Memory
* Data Memory
* Branch and Jump Instructions
